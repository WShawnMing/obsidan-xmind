# Obsidian XMind

一个面向 Obsidian 的思维导图插件方案，目标是在 Obsidian 内提供接近 XMind / 飞书思维导图的节点编辑体验，并支持基于 Markdown 层级自动生成思维导图。

当前仓库先沉淀需求与技术方案，后续可直接按文档进入实现。

- [产品需求文档](./docs/PRD.zh-CN.md)
- [技术设计文档](./docs/TECH-DESIGN.zh-CN.md)

## 当前建议方向

- 第一阶段先做 Obsidian 内可用的最小闭环，而不是一次性追平 XMind 全量能力。
- `Markdown -> 思维导图` 不是附加能力，而是首发核心能力。
- Markdown 是唯一内容真相，不生成独立脑图内容文件；额外只持久化 layout 元数据。
- 节点标题内保留并识别 `[[wiki link]]`，脑图节点天然就是笔记入口。
- 优先保证结构清晰、编辑稳定、同步可预期，再做主题、导出、样式增强。
